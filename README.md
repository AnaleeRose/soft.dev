# soft.dev

A blog web app under developement.


## Tech used:
- React
- Typescript
- Redux
- React-router-dom
- Tailwind
 
## to run the code: 

```bash
git clone https://github.com/ReallyCoolProjects/soft.dev.git
cd soft.dev
npm install
npm run dev 
```

If the page doesn't open automatically, enter this in your browser address : `localhost:3000`
